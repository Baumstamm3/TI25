Mathematischer Rechner

Regeln zur Eingabe

  • Benutze folgende Zeichen zur Darstellung des gewünschten Terms:
    • Addition : [ + ] Bsp.: a + b
    Subtraktion : [ - ] Bsp.: a - b
    Multiplikation : [ * ] Bsp.: a * b
    Division : [ / ] Bsp.: a / b
    Priorisierung : [ () ] Bsp.: (a + b) * c
    Potenzen : [ ^ ] Bsp.: a ^ b
    Fakultät : [ ! ] Bsp.: a!
    Doppelte Fakultät : [ !! ] Bsp.: a!!
    Modulo : [ % ] Bsp.: a % b
    Kommastellen : [ . ] Bsp.: 2.5
  • Die Trennung jedes Elements durch Leerzeichen ist nicht nötig
  • Variablen werden nicht unterstützt.
  • Drücke [Enter], um die Eingabe abzusenden.
